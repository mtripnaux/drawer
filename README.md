# Drawer

Drawer is a mobile frontend for the [tupp](https://github.com/mtripnaux/tupp) contact manager. It operates with a tupp server, which can be run locally or distantly, on your private server for instance. To setup your server, please install tupp, then follow the `tupp serve` procedure by setting up a private password for your API. Once your server is online, you can provide a tupp URI in the application settings.

To run the application, please install the dependencies then run the expo app. You will see a QR code, use the Expo GO mobile aplication to scan it and you will have access to a fully working Drawer. However, this will require you don't kill the npx process. To install a definitive version of Drawer on android, please use the releases tab to get a single `.apk` file.
```bash
npm install
npx expo start
```
